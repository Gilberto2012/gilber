Questão 1 (15 pontos)
Enunciado: Uma máquina térmica opera entre duas fontes, uma quente a Tq = 600 K e uma fria a Tf = 300 K. Em cada ciclo, a máquina absorve Qq = 800 J da fonte quente.

(a) Qual é a eficiência máxima possível dessa máquina? (5 pts)
(b) Se a máquina real tem eficiência de 35%, qual é o calor rejeitado por ciclo? (5 pts)
(c) Qual é a variação de entropia do universo por ciclo, considerando a máquina real? (5 pts)
Passo 1 — Eficiência máxima (Carnot):
ηC = 1 − Tf/Tq = 1 − 300/600 = 0,5 = 50%
Passo 2 — Calor rejeitado (máquina real com η = 35%):
η = 1 − Qf/Qq ⇒ Qf = Qq · (1 − η) = 800 × (1 − 0,35) = 520 J
Passo 3 — Variação de entropia do universo:
ΔSq = −Qq/Tq = −800/600 = −1,333 J/K
ΔSf = +Qf/Tf = +520/300 = +1,733 J/K
ΔSuniv = ΔSq + ΔSf = −1,333 + 1,733 = +0,400 J/K
💡 Comentário: A eficiência máxima é a de Carnot (50%). Qualquer máquina real tem η < ηC. A entropia do universo aumenta (ΔS > 0), confirmando a irreversibilidade.
✅ Resposta:
(a) ηmáx = 50%
(b) Qf = 520 J
(c) ΔSuniv = +0,400 J/K
Questão 2 (15 pontos)
Enunciado: Um mol de um gás ideal monoatômico (CV = 3R/2) sofre uma expansão adiabática reversível de um volume Vi = 2,0 L para Vf = 8,0 L. A temperatura inicial é Ti = 300 K.

(a) Calcule a temperatura final. (5 pts)
(b) Calcule o trabalho realizado pelo gás. (5 pts)
(c) Calcule a variação de entropia do gás. (5 pts)
Passo 1 — Temperatura final (adiabática reversível):
T Vγ−1 = constante, com γ = CP/CV = (5R/2)/(3R/2) = 5/3
Tf = Ti · (Vi/Vf)γ−1 = 300 × (2/8)2/3 = 300 × (1/4)2/3
Tf = 300 × 0,39685 = 119,1 K
Passo 2 — Trabalho (adiabática: Q = 0):
W = −ΔU = −nCV(Tf − Ti) = −(3R/2)(119,1 − 300)
W = −(3 × 8,31/2) × (−180,9) = 2.255 J
Passo 3 — Variação de entropia:
ΔS = 0 (processo adiabático reversível ⇒ isentrópico)
💡 Comentário: Em uma adiabática reversível, Q = 0 e o processo é isentrópico (ΔS = 0). O trabalho é realizado às custas da energia interna.
✅ Resposta:
(a) Tf = 119,1 K
(b) W = 2.255 J
(c) ΔS = 0
Questão 3 (20 pontos)
Enunciado: Dado o potencial termodinâmico F = F(T, V, N) (Energia Livre de Helmholtz):

(a) Escreva a diferencial completa dF. (5 pts)
(b) A partir de dF, obtenha as relações que definem S, P e μ. (5 pts)
(c) Aplique a igualdade das derivadas mistas para obter a relação de Maxwell correspondente. (5 pts)
(d) Para um gás ideal, F = −NRT ln(V/N) + f(T). Calcule P e S. (5 pts)
Passo 1 — Diferencial de F:
dF = −S dT − P dV + μ dN
Passo 2 — Relações para S, P e μ:
S = −(∂F/∂T)V,N
P = −(∂F/∂V)T,N
μ = (∂F/∂N)T,V
Passo 3 — Relação de Maxwell: Igualando as derivadas mistas de F:
(∂S/∂V)T,N = (∂P/∂T)V,N
Passo 4 — Gás ideal: F = −NRT ln(V/N) + f(T)
P = −(∂F/∂V)T,N = NRT/V = nRT/V ✅ (equação dos gases ideais)
S = −(∂F/∂T)V,N = NR ln(V/N) − f′(T)
💡 Comentário: A energia livre de Helmholtz é o potencial natural para processos a T e V constantes. A relação de Maxwell obtida é uma das quatro principais.
✅ Resposta:
(a) dF = −S dT − P dV + μ dN
(b) S = −(∂F/∂T)V,N; P = −(∂F/∂V)T,N; μ = (∂F/∂N)T,V
(c) (∂S/∂V)T = (∂P/∂T)V
(d) P = NRT/V; S = NR ln(V/N) − f′(T)
Questão 4 (15 pontos)
Enunciado: Considere a transformada de Legendre que leva da energia interna U(S, V) para a entalpia H(S, P).

(a) Escreva a definição de H e sua diferencial. (5 pts)
(b) Obtenha a relação de Maxwell a partir de H. (5 pts)
(c) Se H = (5/2)PV para um gás ideal, mostre que CP = 5R/2. (5 pts)
Passo 1 — Definição e diferencial de H:
H = U + PV
dH = T dS + V dP
Passo 2 — Relação de Maxwell: Igualando as derivadas mistas de H:
(∂T/∂P)S = (∂V/∂S)P
Passo 3 — Gás ideal: H = (5/2)PV
Para um gás ideal: PV = nRT ⇒ H = (5/2)nRT
CP = (∂H/∂T)P = (5/2)nR
Para 1 mol: CP = 5R/2 (gás diatômico ideal)
💡 Comentário: A entalpia é o potencial natural para processos a pressão constante. A relação de Maxwell obtida é útil para calcular variações de entropia em processos isobáricos.
✅ Resposta:
(a) H = U + PV; dH = T dS + V dP
(b) (∂T/∂P)S = (∂V/∂S)P
(c) CP = 5R/2
Questão 5 (15 pontos)
Enunciado: Um refrigerador de Carnot opera entre Tf = 250 K e Tq = 300 K.

(a) Qual é o coeficiente de desempenho (CDP) do refrigerador? (5 pts)
(b) Se o refrigerador remove Qf = 500 J da fonte fria por ciclo, qual é o trabalho necessário por ciclo? (5 pts)
(c) Qual seria o CDP se o refrigerador fosse uma máquina real com eficiência de Carnot de 60%? (5 pts)
Passo 1 — CDP de Carnot (refrigerador):
KC = Tf/(Tq − Tf) = 250/(300−250) = 250/50 = 5,0
Passo 2 — Trabalho necessário:
K = Qf/W ⇒ W = Qf/K = 500/5 = 100 J
Passo 3 — CDP real com 60% da eficiência de Carnot:
Kreal = 0,60 × KC = 0,60 × 5,0 = 3,0
💡 Comentário: O CDP de um refrigerador real é sempre menor que o de Carnot. Quanto menor a diferença de temperatura, maior o CDP.
✅ Resposta:
(a) KC = 5,0
(b) W = 100 J
(c) Kreal = 3,0
Questão 6 (20 pontos)
Enunciado: Um sistema tem a seguinte equação fundamental na representação da entropia:
S = A (UVN)1/3
onde A é uma constante positiva.

(a) Verifique se a equação satisfaz a extensividade (Postulado III). (4 pts)
(b) Encontre as equações de estado: 1/T, P/T e μ/T. (6 pts)
(c) Encontre a equação de estado P = P(T, v). (5 pts)
(d) Mostre que este sistema satisfaz a Terceira Lei (Nernst). (5 pts)
Passo 1 — Extensividade: Substitua (U,V,N) por (λU, λV, λN)
S(λ) = A(λU·λV·λN)1/3 = A λ (UVN)1/3 = λ S

✅ A equação é extensiva.

Passo 2 — Equações de estado:
1/T = (∂S/∂U)V,N = (A/3)(VN)1/3 U−2/3
P/T = (∂S/∂V)U,N = (A/3)(UN)1/3 V−2/3
μ/T = −(∂S/∂N)U,V = −(A/3)(UV)1/3 N−2/3
Passo 3 — P = P(T, v):
Em termos molares: u = U/N, v = V/N
1/T = (A/3) v1/3 u−2/3 ⇒ u = (A3/27) v T−3
P/T = (A/3) u1/3 v−2/3
P = T · (A/3) · (A/3) v1/3 T−1 v−2/3 = A²/9 · v−1/3
Passo 4 — Terceira Lei (Nernst):
Da equação de estado: 1/T = (A/3)(VN)1/3 U−2/3
T → 0 ⇒ 1/T → ∞ ⇒ U → 0
Quando U → 0, S = A(UVN)1/3 → 0
✅ Satisfaz a Terceira Lei: S → 0 quando T → 0
💡 Comentário: Esta equação fundamental é um modelo simples que satisfaz todos os postulados termodinâmicos. É um bom exemplo para exercitar a obtenção de equações de estado.
✅ Resposta:
(a) Extensiva ✅
(b) 1/T = (A/3)(VN)1/3U−2/3; P/T = (A/3)(UN)1/3V−2/3; μ/T = −(A/3)(UV)1/3N−2/3
(c) P = (A²/9) v−1/3
(d) S → 0 quando T → 0 ✅
